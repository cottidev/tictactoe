# Tic Tac Toe Arcade

A Tic Tac Toe game with scores and match history.

## The Features are:
* **Scoreboard:** Tracks wins for X and O.
* **Match History:** Records every game result
* **Alternating Turns:** The starting player switches every match X - O ... O - X to be fair :)
* **Responsive:** Works on both desktop and mobile.

## How to Play:
1. Use **Restart** to play the next round.
2. Use **Reset Score** to reset everything to zero.
